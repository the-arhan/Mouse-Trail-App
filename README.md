A Mouse Trail, made by me, ChatGPT and Gemini in an attempt to learn and understand C++! 

I am really sorry, but if you haven't already caught up, **this is only for a Windows machine**. For Linux and Mac, I am getting it ready, so don't much worry! 😉

### ❓ What is this?
This is a lightweight, invisible background app that attaches a smooth, highly customizable trail to your mouse cursor. I built this to learn programming, so it operates as a click-through overlay—meaning it will never interrupt your typing, clicking, or gaming. It just sits quietly in the background and looks awesome while you work.

### ⌨️ Universal Controls
Because there is no traditional window or "X" button to close the app, you navigate it entirely using these global hotkeys:
* `Ctrl + Alt + M` : Cycle through the different visual trail modes
* `Ctrl + Alt + E` : Exit and safely close the application
